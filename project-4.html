CrossKit - IOS APP

Category : UX - UI / Apps
Completion : Nov 2018
Sector : Wellness

This project is an IOS App for crossfiters. Based on a kit with necessary features like weight control, body composition, workouts, mobility testings... It is the beginning of a better user's lifestyle.

Problem & Solution

The problem is that such an advanced sport is not digitized at all . Athletes keep scoring their marks on paper or in the "Leaderboard" but this whole process involves frustrating situations for them.

The proposal is a product that allows the user to keep a record of their workouts, weight control and mobility testing. This way the current frustrations that they face dissappear.

Data reflects the user's need to know their mobility status and how to improve it, to control their personal training marks and the chance to obtain an adapted program in order to improve performance.

First of all I organized data through Blueprint and Lean Ux Canvas so I could set goals and defined a path. To better understand user's need I used tools like Emphaty map. I found several pain points drawing User Journey but most of them were design opportunities and I could make them into great product features.

User Interface

It consists of translating the concepts obtained in the research into a visual and tangible product with the aim of guaranteeing that it has elements of easy access, understanding and use. For this, I combined three fundamental pillars; design, interaction and information architecture.

The Moodboard is an exploratory research tool very useful for the generation of ideas. Through the attributes of the product I relied on the moodboard representing color, shape, texture, tone and illustration.

The font selected for this product was SF Pro, why? because of the user'smental model. I could get from research that more than 76% of people that use fitness Apps are IOS system so using IOS typography would help me designing a "familiar" product.
